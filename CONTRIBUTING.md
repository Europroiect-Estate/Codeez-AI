# Contributing to Codeez

Thank you for your interest in contributing.

## Development

- Go 1.23 or later is required.
- Build: `go build ./cmd/codeez`
- Test: `go test ./...`
- Lint: `golangci-lint run ./...` (when configured)

## Pull requests

1. Open an issue or comment on an existing one to align on scope.
2. Fork the repo and create a branch.
3. Make your changes and ensure `go test ./...` passes.
4. Submit a PR with a clear description and reference to any issue.

## Code of conduct

Please follow our [Code of Conduct](CODE_OF_CONDUCT.md).
